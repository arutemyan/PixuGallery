## プロジェクト概要
イラストポートフォリオサイト（PHP + SQLite）

## 技術スタック
- **バックエンド**: PHP 8.x (PSR-4 autoloading)
- **データベース**: SQLite
- **フロントエンド**: Vanilla JS, CSS
- **画像処理**: GD Library (WebP形式)

## 運用上の注意（タグの入力ルール）

- タグ名およびタグ検索語に以下の文字を含めることはできません: `%`, `_`, `\`。
- 理由: データベースの LIKE 演算子でこれらがワイルドカードやエスケープ文字として解釈され、DB間の挙動差異があるため、内部ツールでは禁止して一律の挙動にしています。
- 実装: 管理画面フロントエンドで即時バリデーションを行い、管理API・公開API（検索）でもサーバー側で拒否して HTTP 400 を返します。

運用担当者は既存データに該当文字を含むタグがないか確認し、もし存在する場合は管理画面で置換または削除してください。
